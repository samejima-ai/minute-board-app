# Context: frontend/data/scenarios.ts

## 目的 (Purpose)

バックエンドの構造化ロジック開発およびデバッグのための、定義済みテスト会話データセット。
マイク入力を使用せずに、一連の会話フローを再現するために使用される。

## データ構造 (Data Structures)

- `SCENARIO_01` (string[]):
  - 定例会議の設定に関する一連の発話リスト。
  - 意図的な「訂正」（時間の変更など）を含み、バックエンドが文脈を正しく理解できるかをテストする。

## 設計判断 (Design Decisions)

- **Static Data**: 変更頻度が低いため、DBではなく静的なTypeScriptファイルとして定義。
